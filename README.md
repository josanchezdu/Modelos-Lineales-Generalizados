# Modelos-Lineales-Generalizados
Análisis estadístico mediante Modelos Lineales Generalizados (GLM) para identificar factores que influyen en el crecimiento del arbolado urbano en Bogotá D.C. Incluye preprocesamiento, filtrado de calidad y análisis sobre una muestra del censo forestal de la ciudad.
